Sobre o projeto
Este projeto busca ajudar pessoas, principalmente os jovens estudantes, a lidarem com a ansiedade e as dificuldades que se encontram na escola, com destaque no ensino médio e no ensino superior, de uma forma leve, divertida e com certa prontidão.

Objetivo
Esta plataforma busca trazer equilíbrio em forma de autoajuda e diversão, através de itens como os diários, check-ins e algumas informações sobre o que é a ansiedade e as formas de lidar com este estado emocional, principalmente em momentos de crise.

Tecnologias utilizadas
Foi utilizado para este projeto:
Node.js para a estruturação da lógica;
Firebase para o armazenamento de dados necessários na nuvem;
Bootstrap, HTML, CSS, entre outros.

Funcionalidades
Pensamos em utilizar, além de algumas idéias básicas sobre neurociência e psicologia, um mascote, para que o usuário se divirta com ele; uma suposta parede para colar post-its com lembretes ou o que o usuário quiser escrever nele; um calendário; e uma página de dicas, para que o usuário saiba lidar com a ansiedade quando ela vier bater à porta; além da central de ajuda, onde está contido os números de telefone para falar com pessoas profissionais na área, em caso de extrema urgência.

Estrutura do projeto
Pensamos o projeto numa organização baseada em MVC (Model, View e Controller), que também contará com outras pastas, como "public" para arquivos estáticos, por exemplo as fotos necessárias para este projeto.

Como executar
Para executar o projeto em Localhost (servidor local - do computador), é necessário seguir as seguintes instruções:

Abrir o projeto;
Em "Terminal", digite: "npm install";
Após isso, digite: "node app.js";
Então, entre no link, que será algo como: "http://localhost300";
Com isso, o projeto estará sendo executado num servidor local, apenas para teste, porém com todas as funcionalidades.

Aplicativo Mobile

Pitch - resumo
Em vista do número de jovens com ansiedade e depressão crescendo a cada dia, o aplicativo Nino busca trazer uma forma de ajuda para os estudantes e jovens encontrarem equilíbrio emocional durante seu dia a dia de uma maneira leve, divertida e organizada, com a possibilidade de uma chamada urgente a algum centro especializado nesse assunto, se necessário.
